## 🎯 QUICK START

### Step 1: Install Dependencies
```bash
cd MYPORTFOLIO/my-portfolio
npm install
```

### Step 2: Start Development Server
```bash
npm run dev
```
Visit: `http://localhost:5173`

### Step 3: Customize Your Information
- Update name and title in `src/components/Hero.tsx`
- Update about text in `src/components/About.tsx`
- Add your projects in `src/components/Projects.tsx`
- Update skills in `src/components/Skills.tsx`
- Update contact info in `src/components/Contact.tsx`

### Step 4: Build for Production
```bash
npm run build
```

### Step 5: Deploy
- **Vercel**: `vercel`
- **Netlify**: `netlify deploy --prod --dir=dist`
- **GitHub Pages**: Push to gh-pages branch
